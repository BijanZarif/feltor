Detailed Description

The Setting for the pipeline.

The Setting consists of parameters and capacities! With this construction plan you can go to the pipeline manufacturer who constructs the pipeline. It is recommended to call

try{ blueprint.consistencyCheck();}
catch( toefl::Message& m){m.display();}

before constructing a Pipeline to catch any Messages before construction.
