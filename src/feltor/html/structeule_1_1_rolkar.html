Detailed Description

template<class Matrix, class container, class Preconditioner>
struct eule::Rolkar< Matrix, container, Preconditioner >

Implicit (perpendicular diffusive) terms for Feltor solver.

\[ \begin{align} -\nu_\perp\Delta_\perp^2 N \\ \frac{C}{\mu} (U_e - U_i) - \nu_\perp\Delta_\perp^2 U \end{align} \]
