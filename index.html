Welcome to the FELTOR project!

For the CPU-Version you'll need to download thrust https://github.com/thrust/thrust and the latest cusp version (the master branch) https://github.com/cusplibrary/cusplibrary

If you want to compile for a GPU you need to install the latest version of CUDA.

If you want to have a nice OpenGL window download the draw library https://github.com/mwiesenberger/draw which needs OpenGL and glfw (cf. README)

If you want to write to the disk install netcdf.

For library documentation see DG lib, FILE lib, TOEFL lib, and SOLOVEV lib.

For source code documentation see FELTOR code, FELTOR2D code.
