Generate the points and the elements of the Jacobian.

Call the width() and height() function before calling this function!

Parameters

eta1done-dimensional list of points inside the eta-domain (0<eta<height())
x= x(zeta,eta)
y= y(zeta,eta)
zetaX= zeta_x(zeta,eta)
zetaY= zeta_y(zeta,eta)
etaX= eta_x(zeta,eta)
etaY= eta_y(zeta,eta)
Note
All the resulting vectors are write-only and get properly resized
