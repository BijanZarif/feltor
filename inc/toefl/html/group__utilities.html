Update a texture with a given field for use of the glTexImage2D() function.

Negative values are blue, positive red. Values near zero are white.

void toefl::gentexture_RGBf_temp ( Texture_RGBf &  tex,
const M &  theta,
const double  ray 
)

Update a texture with a given field (special function for the temperature field)

The maximum of the temperature field is known to be the rayleigh number. T = theta + R(1-z) is taken for the texture, i.e. T = R corresponds to red, T=0 to blue and T=R/2 to white.

Parameters
texA texture array.
thetaThe temperature field
raythe Rayleigh number R
Exceptions
AMessage if tex and theta don't have the same sizes
Note
A colormap is statically constructed and reused at every entry to the function.
