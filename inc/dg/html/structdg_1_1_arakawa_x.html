template<class Matrix, class container>
struct dg::ArakawaX< Matrix, container >

X-space generalized version of Arakawa's scheme.

Template Parameters
MatrixThe Matrix class to use
containerThe vector class on which to operate on. The blas2 function symv( m, x, y) must be callable and may not change x.
