Detailed Description

template<class Matrix, class Vector, class Preconditioner>
struct dg::Helmholtz2< Matrix, Vector, Preconditioner >

Matrix class that represents a more general Helmholtz-type operator.

Unnormed discretization of

\[ \left[ \chi +2 \alpha\Delta + \alpha^2\Delta \left(\chi^{-1}\Delta \right)\right] \]

where \( \chi\) is a function and \(\alpha\) a scalar. Can be used by the Invert class

Template Parameters
MatrixThe cusp-matrix class you want to use
VectorThe Vector class you want to use
PreconditionerThe Preconditioner class you want to use
Attention
The Laplacian in this formula is positive as opposed to the negative sign in the Elliptic operator
