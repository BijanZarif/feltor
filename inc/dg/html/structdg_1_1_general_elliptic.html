Detailed Description

template<class Matrix, class Vector, class Preconditioner>
struct dg::GeneralElliptic< Matrix, Vector, Preconditioner >

Operator that acts as a 2d negative elliptic differential operator.

The term discretized is

\[ -\nabla \cdot ( \mathbf \chi \mathbf \chi \cdot \nabla ) \]

In cartesian coordinates that means

\[ \begin{align} g = \chi_x \partial_x f + \chi_y\partial_y f + \chi_z \partial_z f \\ -\partial_x(\chi_x g ) - \partial_y(\chi_y g) - \partial_z( \chi_z g) \end{align} \]

is discretized while in cylindrical coordinates

\[ \begin{align} g = \chi_R \partial_R f + \chi_Z\partial_Z f + \chi_\phi \partial_\phi f \\ -\frac{1}{R}\partial_R(R\chi_R g ) - \partial_Z(\chi_Z g) - \partial_\phi( \chi_\phi g) \end{align} \]

is discretized.

Template Parameters
MatrixThe Matrix class to use
VectorThe Vector class to use
PreconditionerThe Preconditioner class to use This class has the SelfMadeMatrixTag so it can be used in blas2::symv functions and thus in a conjugate gradient solver.
Note
The constructors initialize \( \chi_x = \chi_y = \chi_z=1\)
Attention
Pay attention to the negative sign
